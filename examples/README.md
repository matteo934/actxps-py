# Examples

This folder will host end-to-end example notebooks demonstrating the package on synthetic data.

Planned examples (per [ROADMAP.md](../ROADMAP.md)):

- `01_synthetic_portfolio.ipynb` — generate a synthetic life insurance portfolio (Phase 1)
- `02_exposure_calculation.ipynb` — compute central exposure with worked examples (Phase 1)
- `03_mortality_study_basic.ipynb` — run an A/E study against SIM92 (Phase 3)
- `04_segmented_study.ipynb` — segment by age, sex, duration; apply credibility (Phase 3)
- `05_full_report.ipynb` — generate a complete HTML report end-to-end (Phase 4)
